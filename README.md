# ⚽ GameNi 90
A football web application that gives users quick access to competition fixtures, league standings, and top scorers — all in one place.

## About
GameNi 90 is a football dashboard named after the iconic 90 minutes of a football match. It allows users to search for any competition and instantly retrieve live fixtures, table standings, and top scorers

## Features

🗓️ Competition Fixtures — Search for upcoming and recent match fixtures by competition name
📊 League Standings — View the current table for any competition (uses short codes like PL for Premier League)
🥅 Top Scorers — Find the leading goalscorers for any competition
🔍 Search functionality across all three sections
✨ Smooth hover animations for an interactive user experience


## Technology Used
HTML5
Tailwind CSS v4 (CDN)
JavaScript (index.js)
Football Data API
